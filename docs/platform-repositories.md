# StoneAI platform repositories

KnowTrail is an independent public product repository. Shared platform code is
owned outside this repository:

| Responsibility | Repository |
| --- | --- |
| Account administration frontend | `laby-ai/stoneai-admin-web` |
| Website and Studio web shell | `laby-ai/stoneai-studio-web` |
| Python AI services and integration tools | `laby-ai/stoneai-ai-python` |
| Account, authorization and billing backend | `laby-ai/stoneai-admin-backend` |

KnowTrail consumes authentication, entitlement and billing contracts from the
backend but retains ownership of research sources, retrieval, grounded
products, artifacts and virtual-classroom integration. The four Gitee projects
remain read-only engineering references; no source is copied into KnowTrail.
